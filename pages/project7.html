XR Pose Estimation

As part of my End-of-Studies Internship, I am currently working on an exciting XR (Extended Reality) project. The goal is to integrate object detection and pose estimation, with a focus on ARUCO markers, into an immersive environment. A key challenge I'm addressing is offloading the heavy computer vision processing outside of Unity. To achieve this, I am developing a system where the camera feed is processed externally using OpenCV, which handles detection and pose estimation before sending the rvec and tvec data back to Unity.

Additionally, I have designed an architecture that supports multi-user functionality. This is done by implementing proxy servers that communicate between each client and their corresponding OpenCV process, all managed by a load balancer to ensure smooth operation. Due to confidentiality agreements, I cannot share further details, but this project is a significant step in advancing my technical expertise in XR development and real-time systems.
